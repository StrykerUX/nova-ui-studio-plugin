Plugin de personalización para WordPress que permite personalizar extensivamente el tema NovaUI con un enfoque en diseño Soft Neo-Brutalist.

== Description ==

**NovaStudio** es un potente plugin de personalización que funciona junto con el tema **NovaUI** para WordPress. Permite a los usuarios personalizar extensivamente todos los aspectos visuales del tema, desde colores y tipografía hasta elementos específicos de la interfaz, manteniendo un enfoque en el estilo Soft Neo-Brutalism con elementos de UI de videojuego indie.

### Características Principales

* **Panel de administración** intuitivo para personalización visual completa
* **Editor CSS avanzado** para personalizaciones específicas
* **Sistema de presets** para configuración rápida de estilos
* **Vista previa en tiempo real** de los cambios
* **Soporte para tema claro/oscuro** con detección automática de preferencias
* **Personalización por plan de membresía** (requiere WooCommerce Memberships)
* **Integración completa** con WooCommerce y extensiones de membresía

### Áreas de Personalización

* **Colores**: Esquemas de color completos con paletas predefinidas
* **Tipografía**: Fuentes, tamaños, estilos y jerarquía tipográfica
* **Componentes**: Estilos de tarjetas, botones, formularios, tablas, etc.
* **Layout**: Personalización de sidebar, header, footer y dashboard
* **Efectos**: Transiciones, animaciones, sombras, bordes
* **WooCommerce**: Personalización de páginas de producto, checkout, etc.

### Niveles de Personalización

El plugin ofrece tres niveles de personalización adaptados a diferentes planes de membresía:

* **Nivel Básico (Para Todos los Usuarios)**
  * Selección de tema claro/oscuro
  * Opciones básicas de colores primarios
  * Personalización de avatar y perfil

* **Nivel Intermedio (Planes Profesionales)**
  * Personalización de componentes específicos
  * Selección entre múltiples presets
  * Ajustes de tipografía y espaciado

* **Nivel Avanzado (Planes Empresa)**
  * Editor CSS completo
  * Personalización completa del dashboard
  * Posibilidad de CSS personalizado por componente
  * Branding completo (logos, colores, fuentes)

### Integración con Plugins Propios

El plugin está especialmente diseñado para integrarse con:

* **Chat IA**: Personalización de interfaces de chat con asistentes IA
* **Quick Links**: Personalización de perfiles de enlaces y estadísticas
* **Futuro CRM**: Estructura base para componentes y vistas de clientes

### Requisitos

* WordPress 5.5 o superior
* PHP 7.4 o superior
* Tema NovaUI instalado y activado
* Navegadores modernos (2 últimas versiones)

== Installation ==

1. Sube los archivos del plugin al directorio `/wp-content/plugins/nova-ui-studio-plugin`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Asegúrate de tener instalado y activado el tema NovaUI
4. Ve a 'NovaStudio' en el panel lateral para comenzar la personalización

== Frequently Asked Questions ==

= ¿Es necesario tener el tema NovaUI para usar este plugin? =

Sí, el plugin NovaStudio está diseñado específicamente para funcionar con el tema NovaUI y no funcionará correctamente con otros temas.

= ¿Puedo utilizar el plugin sin WooCommerce Memberships? =

Sí, el plugin funcionará sin WooCommerce Memberships, pero todas las funciones estarán disponibles en el nivel básico. Para aprovechar las características de personalización por niveles, se recomienda usar WooCommerce Memberships.

= ¿El plugin es compatible con constructores de páginas? =

Sí, el plugin es compatible con la mayoría de los constructores de páginas populares como Elementor, Beaver Builder y Divi. Sin embargo, para obtener la mejor experiencia y coherencia visual, recomendamos utilizar el sistema de shortcodes incluido.

= ¿Cómo puedo personalizar los colores del tema? =

Ve a NovaStudio > Colores en el panel de administración de WordPress. Allí encontrarás opciones para cambiar todos los colores principales, incluyendo presets predefinidos.

= ¿Puedo exportar mis personalizaciones? =

Sí, en la parte inferior de cada página de configuración encontrarás opciones para exportar e importar tus personalizaciones.
